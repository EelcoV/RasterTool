About this tool

This tool is part of the Raster method. For more information, see the Raster project website.

This Raster tool is created using open web technologies including HTML5 and Javascript. It makes use of the JQuery toolkit, together with JQueryUI. Diagrams are created using jsPlumb. In-place editing is done using the jquery-in-place-editor plugin. Some icons obtained from oNline Web Fonts.

This work is sponsored by Radiocommunications Agency Netherlands, and by University of Twente.
