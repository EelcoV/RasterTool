How to use this tool

Short outline of the tasks that you need to perform:

  1. Start by reviewing the default vulnerabilities on Wireless and Wired links and Equipment items.
    Click on the pencil icon on the templates at the top of the screen.
  2. Create a tab for each telecommunication service in your project.
    Use the plus-button at the bottom-left of the screen.
  3. Create the diagram for each telecom service.
    Drag diagram nodes onto the workspace from the templates at the top of the screen. Connect nodes to each other by dragging the endpoint at the top of diagram nodes.
  4. Complete the assessment of single vulnerabilities on components.
    Use the Single failures tab on the left, or the Vulnerabilities menu item on diagram nodes.
  5. Create clusters of components and assess Common cause failures.
    Use the Common cause failures tab on the left.
  6. Review your assessment using the Analysis tab on the left.

For more information, see the manual.
